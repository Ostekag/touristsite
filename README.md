# Туристический сайт города Астана


**Тема:** Туристический сайт города


Проект представляет собой многостраничный информационный сайт, посвящённый городу Астана. Сайт содержит основную информацию о городе, его достопримечательностях и контактную страницу.


## Страницы сайта
- Главная
- О городе
- Достопримечательности
- Контакты


## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid)
